QUICK START — GET THE MOST FROM PRO
--------------------------------------------------------------------------------

ADD A SONG WITH LOCAL AUDIO (RECOMMENDED)
  Library → Add song → Upload audio file
  Local files unlock offline playback, auto-align, speed control, A/B export,
  and vocal separation on capable devices.

FETCH OR PASTE LYRICS
  Utasync searches LRCLIB and YouTube captions automatically. You can also
  paste lyrics or import LRC / SRT / VTT files.

SYNC TIMING
  • Tap-to-sync while the song plays, or
  • Run AI auto-align (requires local audio; downloads a model on first use)

STUDY
  • Tap any line to seek
  • Toggle romaji, furigana, or translation in the display menu
  • Set A/B loop points to repeat a section
  • Slow down with pitch-preserved speed control
  • Try Cloze mode to hide words on the active line

INSTALL AS AN APP (OPTIONAL)
  In Chrome or Edge: browser menu → Install Utasync / Add to Home Screen.
  The app works offline after models and audio are cached.
